Questions

Quick answers

How much does Stack the Lineup cost?
Stack the Lineup is free to download and use for a single team. Stack the Lineup Pro is a one-time $4.99 in-app purchase that unlocks Auto-Fill, position preferences, full game history, multiple teams, and the complete Coaches Guide PDF. There is no subscription.
Does it work for softball too?
Yes. Stack the Lineup works for youth baseball and softball. The positions, fair-play rules, and rotation logic apply to both, and it's used by coaches across Little League, Cal Ripken, Babe Ruth, JBO, travel, and rec leagues.
What is "fair play" and how does the app help?
Fair play means every player gets balanced time across positions and innings, instead of the same kids always sitting or always playing the infield. Stack the Lineup flags missing infield innings, back-to-back bench, catcher-to-pitcher violations, and overuse so you can fix them before the game.
Which devices does it run on?
Stack the Lineup is a native app for iPhone and iPad. It is not currently available on Android or the web.
